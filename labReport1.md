## cd Examples
*no arguments*
```
[user@sahara ~]$ cd
[user@sahara ~]$
```
*path to a directory*
```
[user@sahara ~]$ cd lecture1
[user@sahara ~/lecture1]$
```
*path to a file*
```
[user@sahara ~/lecture1]$ cd Hello.java
bash: cd: Hello.java: Not a directory
```
